Computer Based Test

A decade of paper based examination experience has led the foundation of online assessment, thus creating a robust and scalable examination solution that touches all the aspects of Computer Based Test. The solution is feasible in entrance exams, recruitment tests, mock tests for coaching institutes and national olympiads. SEG SKILLS Computer Based Test provides end to end solution, starting from application registration, center allocation, generating hall tickets, question paper creation, test on computer and finally result declaration.

Computer Based Test Solution

Application Registration
The online application registration captures all the information from the applicant, stores them securely & centrally on a web server. The solution is capable of handling large number of request at same time and registration happens swiftly on the body official website.

Center Allocation
After completion of application registration, center allocation is done automatically with the preferred choice of location as per availability of seats.

Generating Hall Tickets/Admit Cards
Hall Tickets/Admit Cards are generated after center allocation, an email and SMS notification is send to candidates about the availability of Hall Tickets/Admit Card on the web. Candidates can download Hall Tickets/Admit Card from web or email.

Question Paper Creation
Question paper can be uploaded in the server in secure manner by authenticated person. The uploading can be done from a Word/Excel file in few seconds. Not only this, test paper could also be prepared in a collaborative manner where different faculties can enter their subject/topic area questions. A final question paper is automatically created randomly from the pool of questions.

Test on Computer
Question paper is downloaded in secured encrypted format on test center local server from the main server, before start time of the test. The test client is connected to local server and does not require Internet connectivity. The test runs in complete secured manner, having all modern security measures like biometric, screen locking, random photo capturing of candidates etc. The test can resume in-case of power failure or any hardware problem on different computer from where it was disconnected.

Result Declaration
Computer Based Test can be taken in multiple shift, thus the result is declared by using Equating methodology for comparable scores on different sets of exam. The final result is declared on official website of organization. Candidates also get email & SMS notification on their registered email & mobile.
